P.A.R.K


The main objective of this proyect was to create a C++ Game Engine which gives the user all kind of options (not just to create amusement parks games ). In order to test the engine Pure Accurate Roaller Koaster was created, a managment game where you put yourself in the shoes of a business man. You must pay taxes and build in order to satisfy the visitors.

This engine was created using:

  • Ogre3D as Render Engine
  • Bullet as Physics System
  • CEGUI for HUD
  • Json For Modern C++ for data managment
